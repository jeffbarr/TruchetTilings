# Introduction

This repository contains [OpenSCAD](https://openscad.org/) code to generate rectangular grids of hexagons in a form similar to those found in [Kerry Mitchell](https://www.kerrymitchellart.com/)'s 2020 paper, [Generalizations of Truchet Tiles](https://archive.bridgesmathart.org/2020/bridges2020-191.pdf). In that paper, Kerry presented a seemingly simple set of patterns that can be combined in presudo-random or specified ways to produce interesting patterns:
